# PrepVault - Interview Preparation Workspace

PrepVault is an all-in-one platform designed to help students and professionals master technical interviews. It provides a beautiful workspace to organize subjects, take notes, manage projects, save resources, and practice with mock interviews.

## ✨ Features

- **📚 Subjects** - Organize your interview subjects with topics and notes
- **📝 Notes** - Create and manage notes with inline code editing
- **💻 Projects** - Showcase your work with project details and links
- **🔗 Resources** - Save and organize learning resources from various platforms
- **🎯 Mock Interview** - Practice with AI-powered interview questions
- **📅 Revision Planner** - Plan and track your revision sessions
- **👤 Profile** - Manage your personal information and settings

## 🛠️ Tech Stack

- **Frontend:** React 19, Vite 6
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Icons:** Lucide React, React Icons
- **Routing:** React Router DOM 7

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/prepvault.git

# Navigate to the project
cd prepvault

# Install dependencies
npm install

# Start development server
npm run dev
